# Maven Market Sales & Returns Analysis

## Overview
Interactive Power BI dashboard analyzing **$1.76M revenue** and **$1.05M profit** across **10,281 customers** and **3 countries** (USA, Canada, Mexico).

## Key Insights
- USA drives **67%** of total revenue
- **Hermanos** is the top-performing brand (25K units sold)
- **Supermarket** store type generates the highest quantity (373K)
- Customer activation rate: **86%**
- Return rate tracked and analyzed by product brand, card type, and quarter

## Dashboard Pages
1. **Main Page** - Executive overview with KPIs
2. **Sales** - Revenue trends, geographic analysis, product performance
3. **Customers** - Customer segmentation, demographics, member cards
4. **Products** - Top brands, store types, quarterly trends
5. **Returns** - Return quantity, return rates, cost analysis
